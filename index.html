Посилання на gallery.html: перейти
  • git clone [url-адреса] - cкачування репозиторію до комп'ютера;
  • git status - виявлення статусу файлів (новий / змінений / видалений тощо);
  • git add [файл] - збереження змін проекту, що увійдуть до наступного коміту;
  • git commit -m "[коментар для коміту]" - локальна фіксація змін у коміт;
  • git push - завантаження коміту у репозиторій;
  • git branch [ім'я гілки] - створення нової гілки в репозиторію;
  • git log - перегляд історії змін у файлах;
  • git remote -v - перегляд всіх наявних підключених репозиторіїв;
  • git checkout - перемикання між гілками в репозиторію.
